Условный конструктор

- в Python осуществляется с использованием ключевых слов if, elif (необязательно) и else.

Синтаксис следующий :

  1. Примеры работы с целыми числами в условном операторе :

    • В данном примере, если значение переменной число больше нуля, то будет выполнен блок кода внутри условия, и будет выведено сообщение "Число положительное". Условные операторы в Python используются для выполнения различных блоков кода в зависимости от того, выполняется ли указанное условие (в данном случае, число больше 0) или нет.

  2. Примеры работы с условием if-elif :

    • В этом примере, если число больше 0, выполнится первый блок кода. Если число равно 0, выполнится блок кода после elif. В противном случае, если оба предыдущих условия не выполнены, выполнится блок кода после else. Конструкция elif позволяет проверить несколько условий последовательно.

  3. Примеры работы с конструкцией elif :

    • В этом примере, если возраст меньше 18, выполнится первый блок кода. Если возраст от 18 до 64 лет, выполнится блок кода после elif. Если ни одно из предыдущих условий не выполнено, выполнится блок кода после else. elif используется для проверки дополнительных условий, если предыдущие не были выполнены.

  4. Примеры работы с оператором else :

    • В этом примере, если is_adult равно True и has_subscription также равно True, выполнится первый блок кода. Если is_adult истинно, но has_subscription ложно, выполнится блок кода после elif. Если ни одно из условий не истинно, выполнится блок кода после else. Оператор else используется для выполнения кода, если ни одно из предыдущих условий не было выполнено.

Предположим, у вас есть переменная temperature, которая представляет собой температуру в градусах Цельсия. Ваша задача написать программу, которая выводит соответствующее сообщение в зависимости от значения температуры :

  1. Если температура больше или равна 30 градусам, выведите сообщение "Жарко".

  1. Если температура между 20 и 29 градусами включительно, выведите сообщение "Приятная температура".

  1. Если температура меньше 20 градусов, выведите сообщение "Прохладно".
