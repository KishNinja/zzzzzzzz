Переменные в Python

- это именованные места для хранения данных. Типы данных определяют, какие виды данных могут храниться в переменных. Примеры типов данных в Python: int (целые числа), float (числа с плавающей точкой), str (строки), bool (булевы значения).

    • В этом коде создаются две переменные age и quantity, которые хранят целые числа 25 и -10 соответственно. В Python тип данных для целых чисел называется int.

  2. Примеры работы с типом данных float :

    • В данном коде создаются две переменные price и pi, которые хранят числа с плавающей точкой 19.99 и 3.14159 соответственно. Тип данных для таких чисел в Python называется float.

  3. Примеры работы с типом данных str :

    • В этом коде создаются две переменные name и message, которые хранят строки "John" и 'Привет, мир!' соответственно. Тип данных для строк в Python называется str.

  4. Примеры работы с типом данных bool :

    • В этом коде создаются две переменные is_adult и has_subscription, которые хранят булевы значения True (истина) и False (ложь) соответственно. Тип данных для булевых значений в Python называется bool.
