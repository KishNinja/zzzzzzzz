List(Список) в Python

- это удобная структура данных, которая позволяет хранить упорядоченные коллекции элементов. Списки могут содержать объекты любого типа, в том числе другие списки. Они изменяемы, что означает, что вы можете добавлять, удалять и изменять элементы в списке.

Синтаксис следующий :

  1. Примеры создания списка :

  2. Некоторые основные операции со списками:

    • Добавление элемента в конец списка: my_list.append(6)

    • Вставка элемента на определенную позицию: my_list.insert(2, 'new_element')

    • Удаление элемента по значению: my_list.remove('four')

    • Получение индекса элемента: index = my_list.index(3)

    • Изменение значения элемента по индексу: my_list[0] = 'one'

    • Удаление элемента по индексу: del my_list[1]
