Функции в Python

- это блоки кода, которые могут быть определены для выполнения конкретной задачи. Функции используются для организации кода и повторного использования. Они позволяют группировать набор инструкций в один блок, который может быть вызван из разных частей программы.

Примеры :

  1. Примеры функциии цикла for для приветствия нескольких имен :

    • В этом примере функция приветствие принимает один аргумент - имя, а затем цикл for используется для итерации по списку имен и вызова этой функции для каждого имени.

В этом примере у нас есть функция с именем приветствие, которая принимает аргумент имя и выводит приветственное сообщение с использованием переданного имени. Функция определяется с использованием ключевого слова def, за которым следует имя функции и список параметров в скобках. Тело функции начинается с отступа и содержит инструкции, выполняемые функцией.

Вызов функции происходит путем написания ее имени с аргументами в скобках. В данном случае, приветствие("Ваше Имя") вызовет функцию с передачей ей строкового аргумента "Ваше Имя".
