Циклы в Python

- представляют собой конструкции, которые позволяют выполнять определенный блок кода многократно в зависимости от условия. Они позволяют автоматизировать выполнение однотипных действий или обработку данных.

Примеры :

  1. Пример цикла for :

    • Этот код выведет числа от 0 до 4 включительно, так как функция range(5) создает последовательность чисел от 0 до 4. В каждой итерации цикла переменная i принимает значение из этой последовательности, и затем выводится с помощью функции print().

  2. Примеры работы цикла while :

    • Этот код создает цикл while, который будет выполняться, пока переменная count меньше 5. В каждой итерации цикла значение count выводится с помощью функции print(), и затем count увеличивается на 1. Цикл будет повторяться, пока условие count < 5 истинно.
